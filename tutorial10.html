model 1: $$\beta_{10} + \beta_{11} \exp(-\beta_{12}t)$$

$$ \beta_{10} = 1668.011; \beta_{11} = 3217.617; \beta_{12} = 0.026$$

model 2: $$\beta_{20} + \frac{\beta_{21}\exp(-\beta_{22}t))}{(1 + \exp(-\beta_{22}-t)}$$

$$\beta_{20} = 1717.925; \beta_{21} = 3477.049; \beta_{22} = 0.029$$

model 3: $$\beta_{30}(1 - \exp(\beta_{32}-t)) + \beta_{31}\exp(-\beta_{32}t)$$

$$\beta_{30} = 1693.768; \beta_{31} = 3303.327; \beta_{32} = 0.027$$

PI simulation using MC Simulation¶

  • Draw a unit circle within the a unit square
  • Generate random numbers and count how many are in the circle

  • Area of the circle = $\pi r^2$

  • Area of the square = $(2r)^2 = 4r^2$

  • Ratio 1: $\frac{\pi r^2}{4r^2} = \frac{\pi}{4}$

  • Ratio 2: No. points in circle / No. points in the square

Thus:

  • $\pi = 4 \times$ (No. points in circle / No. points in the square)

import matplotlib.pyplot as plt
import numpy as np
import numpy.random as npr

import matplotlib.animation as animation
from IPython.display import HTML

x = np.linspace(0,1,101)
y = np.sqrt(1-x**2)
fig, ax = plt.subplots()
ax.plot(x,y)

# animation function 
def animate(i): 
    xr = npr.uniform(size=1)
    yr = npr.uniform(size=1)
    incirc = xr**2+yr**2<1
    ax.plot(xr[incirc],yr[incirc],"b*")
    ax.plot(xr[np.logical_not(incirc)],yr[np.logical_not(incirc)],"ro")
anim = animation.FuncAnimation(fig, animate, interval=1000) 
HTML(anim.to_html5_video())
